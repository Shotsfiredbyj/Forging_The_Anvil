# Research

External research and learnings that informed Cold Anvil's design. Competitor analysis, LLM-as-judge limits, pricing research, GTM research, session retrospectives.

**The current product spec is at `/home/jack/Forging_The_Anvil/respec/03-spec.md`.**

Files in this directory are evidence, not directives. They are the *reason* the spec is shaped the way it is, but they are not themselves the spec. When a file here conflicts with the spec, the spec wins.

Every major finding that ended up shaping the respec is cited inline in `respec/01-journey.md`. Read that first to see which files were load-bearing and which were reference-only.

Research directory is **kept in place, not archived.** New research on topics that continue to be relevant (pricing iteration, competitor updates, model benchmarks) belongs here. Research that describes product architecture decisions should go into the new implementation's decision log, not here.
